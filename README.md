# EcoMap E2E Testing with Cypress

## Launching Cypress
In your CLI, type in:
`npx cypress open`

## All commands start with
`cypress run [options]`
_[replace [options] with...](https://docs.cypress.io/guides/guides/command-line)_
- `--spec` </br >
- `--headless`
- `--no-runner-ui`
